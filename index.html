Countdown Timer

Description:

Developed a dynamic countdown timer using HTML, CSS & JavaScript that displays the time remaining until a specified end date and time. The timer updates in real-time and presents the remaining days, hours, minutes, and seconds in designated input fields on a webpage.

Key Features:

Date Handling: Utilized JavaScript's Date object to calculate the difference between the current time and the target date.
Dynamic Updates: Employed setInterval to refresh the countdown every second, ensuring real-time accuracy.
User Interface Integration: Synchronized the calculated time with input fields to provide a live countdown display
